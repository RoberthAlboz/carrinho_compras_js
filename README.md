## 💻 Sobre o Projeto

O **Carrinho de Compras JS** é uma aplicação dinâmica desenvolvida para praticar a lógica de programação e a manipulação do **DOM (Document Object Model)**. O projeto permite que o usuário adicione produtos a um carrinho, ajuste quantidades e visualize o cálculo total de forma automática.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**.

---

## ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, foram aplicados os seguintes conceitos técnicos:

-   **Manipulação de DOM:** Criação dinâmica de elementos HTML para exibir itens no carrinho e atualizar valores em tempo real.
-   **Lógica de Programação:** -   Uso de arrays e objetos para gerenciar o estado dos itens.
    -   Estruturas de repetição para renderização da lista.
    -   Condicionais para regras de negócio (como remover o item se a quantidade chegar a zero).
-   **Interatividade:** Eventos de clique para adicionar, aumentar, diminuir e remover produtos.
-   **Cálculos Automáticos:** Atualização instantânea do subtotal e total conforme a interação do usuário.
-   **Versionamento:** Controle de versão utilizando **Git** e hospedagem no **GitHub**.
